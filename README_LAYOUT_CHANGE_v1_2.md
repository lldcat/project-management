# v1.2 布局调整说明

本版本重新调整了小程序的信息架构，避免将业务操作入口直接放在首页。

## 底部导航

调整为 4 个 Tab：

1. 首页：综合看板
2. 项目：正式项目列表、筛选、查看与复核
3. 业务：按角色显示业务操作入口
4. 我的：当前用户、openid、角色与环境信息

## 首页

首页定位为综合看板，不再放置创建或处理入口。

首页展示：

- 项目总数
- 风险项目
- 成本异常
- 进度滞后
- Pre-cal 可见总数
- Sales 已提交数量
- 待绑定 SAP 数量
- 已绑定 SAP 数量
- 关键项目指标汇总
- 风险项目列表

## 项目页

项目页定位为正式项目数据查看与复核页。

- 移除了顶部“新增”按钮
- 创建项目入口移至“业务”页
- 保留项目筛选、查看、编辑、删除和 CSV 导出

## 业务页

新增“业务”Tab，按角色显示不同入口：

- PM / Leader / Admin：创建项目、项目列表
- Sales：新建 Pre-cal、我的 Pre-cal
- CS / Admin：SAP 绑定
- Admin：Pre-cal 管理、参数维护、用户身份

## 我的页

原“设置”Tab 改为“我的”，用于查看 openid、角色和云环境，并刷新登录身份。
